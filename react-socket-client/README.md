# Primeiro passo:

`npm install` - instala as dependências

# Importante:

Se estiver rodando em windows: `npm start`
Se rodar em ubuntu ou MAC OS: altere o script "start" do package.json para `export PORT=6969 && react-scripts start`,
e então, `npm start`
